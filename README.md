# ontology-cyber

This repository is used for collaborative ontology development in Protégé.

## Structure

- `ontology/axioms.ttl` → main ontology file
- `.github/workflows/ontology-ci.yml` → GitHub Actions workflow for basic validation

## Collaboration workflow

1. Create a branch
2. Edit ontology in Protégé
3. Save changes
4. Commit and push
5. Open pull request
6. Review and merge
